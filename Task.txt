
задача:
Сделать сервис, который возвращает заголовок случайной новости из Top 500 с ресурса https://news.ycombinator.com. Заголовки не должны повторяться.


api ресурса:
- https://hacker-news.firebaseio.com/v0/topstories.json для получения id топ500 историй
- ответ списком id
- https://hacker-news.firebaseio.com/v0/item/{$id}.json для получения данных истории
- в ответе есть title

обработка ошибок: 
-получение ответов от ресурса должно быть корректны
